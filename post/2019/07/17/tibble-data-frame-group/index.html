gaoch

2019-07-17

Categories: R Tags: tibble data.frame

Tibble е’Ң Data.frame дёҖиҲ¬еҸҜд»ҘйҖҡз”ЁпјҢдҪҶжҳҜ Tibble дёӯеҗ«жңүзҡ„еҲҶз»„дҝЎжҒҜпјҢеҰӮжһңдёҚжіЁж„ҸпјҢжңүж—¶еҸҜиғҪдјҡеј•иө·й”ҷиҜҜгҖӮ

жңүдёҖдёӘtibbleпјҢgroup by enzymeе’ҢnpsгҖӮ

дҪҝз”Ёcompare_mean()ж—¶дјҡжҠҘй”ҷгҖӮ

> compare_means(Error_Rate ~ enzyme,error_rate_summary,group.by = "type",paired = F,ref.group = ".all.")
Adding missing grouping variables: `nps`
Error in mutate_impl(.data, dots) : 
  Column `.group.` must be length 3 (the group size) or one, not 81
