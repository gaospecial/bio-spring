Apache访问控制, mainfile.php@xoops等等

gaoch

2007-12-02

Tags: 百度空间

xoops安装完之后,从别的电脑上访问,都会定位到http://localhost/*/,根本不能用.今天终于发现了一个解决方法.
修改mainfile.php即可:
找到
define (‘XOOPS_URL’,‘http://localhost/xoops’)
将其改为
define (‘XOOPS_URL’,‘http://{你的域名或者IP}/xoops’)
这样就可以啦.

apache的访问控制
编辑/etc/apache2/sites-available/default文件
在<Directory /var/www/></Directory>块里面,加入
                Order deny,allow
                deny from all
                allow from 123.123
                allow from 127.0.0.1
注释:allow from后面的123.123可以允许这样的一个IP域的访问.

另外,关于重定向,将Redirect 后面设为你想要的目录即可.
